# Cognition rosetta v1  
  
**ID: CZ1_01**  
**STATEMENT: It is forbidden for any cognitive interpretation to modify, weaken, override, or constrain any kernel rule or kernel admissibility condition.**  
**FORBIDS: kernel modification via cognition overlay, cognition-overlay-gated admissibility, cognition-overlay-gated rejection, cognition-defined kernel exceptions**  
**PERMITS: removable cognitive annotations, optional cognitive labels with no kernel force**  
**OPEN: minimal audit criteria for non-binding cognitive overlays**  
  
**ID: CZ1_02**  
**STATEMENT: It is forbidden to treat any cognitive interpretation as true, correct, normal, or preferred by default.**  
**FORBIDS: truth-by-cognition-overlay, correctness-by-cognition-overlay, normality-by-overlay, preferred-interpretation-by-default**  
**PERMITS: multiple optional cognitive lenses with no default correctness status**  
**OPEN: admissible non-binding confidence annotations for interpretations (if any)**  
  
**ID: CZ1_03**  
**STATEMENT: It is forbidden to treat cognitive terminology as granting semantic force to kernel tokens beyond what is explicitly declared within cognition overlay declarations.**  
**FORBIDS: implicit semantics from cognitive labels, meaning inflation by naming, semantic completion by overlay**  
**PERMITS: explicit non-binding cognitive-meaning declarations scoped to overlay tokens**  
**OPEN: minimal schema for explicit non-binding interpretation declarations**  
  
**ID: CZ2_01**  
**STATEMENT: It is forbidden to conflate cognition-overlay namespaces with kernel namespaces; cognition-overlay tokens must not be treated as kernel tokens.**  
**FORBIDS: namespace conflation, cognition-token substitution for kernel tokens, implicit kernel re-tokenization via cognitive terms**  
**PERMITS: explicit cognition overlay namespaces, explicit mapping tables from kernel tokens to cognitive tokens**  
**OPEN: minimal namespace declaration schema**  
  
**ID: CZ2_02**  
**STATEMENT: It is forbidden to assert any cognition-overlay mapping without an explicit mapping declaration specifying source token(s), target cognitive token(s), and scope.**  
**FORBIDS: implicit mappings, scope-free mapping assertions, unstated targets, context-inferred mappings**  
**PERMITS: explicit mapping declarations with explicit scope**  
**OPEN: minimal mapping declaration schema**  
  
**ID: CZ2_03**  
**STATEMENT: It is forbidden to assume cognition-overlay mappings are injective, surjective, invertible, or canonical by default.**  
**FORBIDS: one-to-one assumptions, onto assumptions, invertibility-by-fiat, canonical mapping by default**  
**PERMITS: partial mappings, one-to-many mappings, many-to-one mappings, multiple coexisting mappings**  
**OPEN: admissible premises for partial inverses (if any)**
