3.2 Szilard Engine Mechanics Integration (New Section) 
The Szilard engine (1929) demonstrates information-entropy equivalence: a demon 
measures a particle in a box, inserts a partition, and lets expansion do work kT ln 2, but 
measurement/erasure costs balance it (Landauer's principle). 
Mapping to Model: 
• 
⚫ Bit-0 as Szilard Battery: = kT ln 2 per bit; measurement (Ti(-)) extracts work by 
reducing S(p), increasing J(p). 
• Polarity (Axis 0): N (North/+ ascent) = work extraction (demon's gain); S (South/- 
descent) = erasure cost (entropy dump). 
• 
· 
Toy Math: For 1-bit system (d=2), max J(p) = \log 2. Measurement: p → |0><0| (pure), 
AS-\log 2; work = T AS. 
Type Flows: Inward (Type 1) = measurement/compression (demon inserts partition); 
Outward (Type 2) = expansion/erasure (gas pushes piston). 
• Finitist Proxy: Discrete box with W=4 states; "demon" as probe P measuring position, 
updating p to delta function. 
This ties thermodynamics to IGT: "Win" quadrants extract work from info; "Lose" dump 
entropy.
