**What Carnot actually has (correct)**  
  
A Carnot cycle consists of **four strokes**:  
	1.	Isothermal expansion (at hot reservoir)  
	2.	Adiabatic expansion  
	3.	Isothermal compression (at cold reservoir)  
	4.	Adiabatic compression  
  
Key facts:  
	•	**Hot and cold reservoirs are locations**, not directions.  
	•	**Heating vs cooling** is determined by **cycle direction**.  
	•	Reversing the traversal direction turns an engine into a refrigerator.  
  
So the only binary distinction at the cycle level is:  
  
**forward vs reverse traversal of the same loop**
