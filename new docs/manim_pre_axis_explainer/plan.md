# Pre-Axis Ordering: Why Axis 0 Is Still Embargoed

## Overview
- Topic: the pre-Axis tier ladder and why the system must close lower tiers before Axis 0
- Hook: "Why can’t we jump straight to Axis 0?"
- Aha moment: Axis 0 is not a starting point; it is the consequence of a completed chain from constraints to kernel
- Target audience: technical readers who can follow a layered system diagram, but do not need the full repo context
- Length: ~35–45 seconds
- Resolution: 480p draft, 1080p final
