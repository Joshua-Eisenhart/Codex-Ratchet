# Probe_Induced_Equivalence
**Node ID:** `A2_3::SOURCE_MAP_PASS::Probe_Induced_Equivalence::52aac6fcd2eb74ac`

## Description
Identity is not primitive — it is earned through indistinguishability under probes. a=a iff there is no probe that can distinguish a from itself. This is a nontrivial claim about finite probe families, not a tautology.
