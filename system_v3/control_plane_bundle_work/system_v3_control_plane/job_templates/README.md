# ZIP_JOB Templates

These templates are for upper-layer ZIP_JOB / ZIP subagent bundles.

They are not transport ZIP templates.

Use them for:
- A2 extraction jobs
- A2 to A1 distillation jobs
- A1 generation jobs
- A1 consolidation / prepack jobs
- audit / validation jobs

Transport ZIPs remain under:
- `../templates/`
