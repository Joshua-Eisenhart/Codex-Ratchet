# CROSS_BASIN_REQUIREMENT v1

Purpose: define doctrine for cross-basin exploration fairness in A1 outputs.

This document is doctrine-only.
It does NOT modify `A1_STRATEGY_v1` schema.
It does NOT modify kernel behavior.

## Requirement

- A1 must generate structurally distinct alternatives.
- At least one cross-family candidate must be present at non-zero deterministic rate.
- Enforcement layers must remain neutral across candidate families.
- No probabilistic selection is allowed.

## Scope Constraints

- No schema changes are introduced.
- No new ZIP types or container primitives are introduced.
