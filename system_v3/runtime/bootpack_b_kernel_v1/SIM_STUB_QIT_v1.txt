SIM_STUB_QIT_v1

This file is a deterministic, version-pinned placeholder for the SIM_CODE_HASH_SHA256
field used by MATH_DEF items in bootpack_b_kernel_v1.

It is NOT executed.
It exists so that A1/A0 can reference a non-placeholder, reproducible sha256 for
simulation "code identity" without introducing new container primitives or new
runtime directories.

The ratchet treats all semantics as hostile input; any "meaning" is adjudicated
only by deterministic kernel + SIM evidence rules.
