## 720° Spinor Cycle
- 360° → |Ψ⟩ → -|Ψ⟩ (inverted with Berry phase). NOT at origin.
- 720° → -|Ψ⟩ → |Ψ⟩ (true identity restored, thermodynamic ledger balanced)
- Major Loop (first 360°): establishes structural constraints, accumulates negentropy, locks phase shift
- Minor Loop (second 360°): spends accumulated phase to extract work, produces entropy, returns to true zero

## Winding Limit Saturation
- Single engine on twisted fibers → winding number accumulates → phase space saturates → lockup = Thermal Death
- Prevention: dual counter-rotating conjugate engines
- LEFT_WEYL_CONVERGENT: converges + winds fiber inward (consumes gradient, builds structure)
- RIGHT_WEYL_DIVERGENT: diverges + unwinds fiber outward (restores gradient, produces work)
- Stacking winding + unwinding = non-equilibrium steady state without topological collapse

## Thermodynamic Gate (Bit 1) — Isothermal vs Adiabatic

### Isothermal (Coupled/Diathermal) — Se and Ni topologies
- Boundary OPEN, Q ≠ 0, free heat exchange with bath
- Modeled as open quantum system: CPTP Lindbladian evolution
- Function: READ/WRITE — measure reality, extract external data, vent waste heat
- Pays irreversible Landauer cost

### Adiabatic (Insulated) — Ne and Si topologies
- Boundary CLOSED, Q = 0, no heat exchange
- Modeled as reversible Unitary evolution: UρU†
- Function: COMPUTE/SIMULATE — process logic, consolidate memory, evaluate superpositions
- No immediate thermodynamic penalty
