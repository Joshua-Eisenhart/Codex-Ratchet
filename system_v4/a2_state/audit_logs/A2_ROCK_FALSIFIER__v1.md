# A2 FUEL: THE ROCK FALSIFIER
**STATUS:** A2 HIGH-ENTROPY FUEL — SOURCE-BEARING EXTRACT

## The Rock (Low-Action Genotype)
- NOT a frozen classical state — a "low-action-algebra genotype"
- Small operator set A_c: few allowable unitaries/CPTP channels
- Near-identity channel: E_agent ≈ I
- Pays near-zero Landauer/erasure costs
- No 8-stage cycle, passive homeostasis only

## Environmental Shocks (CPTP Perturbations)
- Sequence: ρ_{t+1} = E_agent(E_shock(ρ_t))
- Shock types: amplitude damping, phase flips, random unitaries U_shock
- Drawn from distribution of environment labels e_t ∈ {1,...,m}
- Must include REGIME SHIFTS: sudden macro-level changes in shock distribution

## Performance Metrics
- **Solvency:** stall projector Π_dead, alive projector Π_A
  - Solvency_T = Tr(Π_A ρ_T) — alive probability at horizon T
  - If Tr(Π_dead ρ_t) ≥ θ → agent dies
- **Ratchet Monotone:** R_T = -log P(stall by T)
- **Competence:** C_T = W_useful / (1 + K) — work extracted vs winding debt

## The Crossover Point
- Short/stationary environment: ROCK WINS (engine wastes energy on complexity)
- Volatile/regime-shifted: ENGINE WINS (builds error-correcting codes for protection)
- Crossover = when environmental volatility exceeds rock's passive absorption capacity

## Kill Conditions for Graveyard
1. **Degeneracy Falsifier:** Rock outperforms engine EVEN under broad volatile shocks → entire solvency→complexity hypothesis KILLED
2. **Complexity Bias Kill:** Engine wins in simple environments → simulation biased, costs miscalculated
3. **Non-Ratchet Falsifier:** R_T or C_T merely oscillates, doesn't trend upward → complexity ratchet hypothesis dies
4. **Anthropic Triviality:** Engine survives only because sim conditioned to require observers → teleology collapses to selection bias
