## The Problem (Bridge T Failure)
The Codex Ratchet theory posits that structural complexity is a thermodynamic liability in calm environments. Therefore, a simple near-identity channel ("The Rock") should out-survive the 8-Stage Process Cycle in low-volatility regimes because the Rock does not pay the massive thermodynamic maintenance cost required to execute 8 active CPTP structural maps. 

However, our simulation (`rock_falsifier_results.json`) shows the Process Cycle winning at `volatility=0.05` (calm environment), surviving with a solvency of **0.344** vs the Rock's **0.047**. This triggers the `E_SIM_ROCK_FALSIFIER_COMPLEXITY_BIAS` token. 

## The Root Cause
The failure fundamentally lies in our math for `compute_solvency(rho, d)`:
```python
def compute_solvency(rho: np.ndarray, d: int) -> float:
    max_S = np.log2(d)
    S = von_neumann_entropy(rho)
    solvency = 1.0 - (S / max_S)
    return max(0.0, solvency)
```

The system equates "Solvency" purely with terminal Structure Gained (`1.0 - (S / max_S)`). 
It **completely ignores** the Landauer limit erasure costs (Entropy Expelled into the bath) required to run the 8 CPTP operators. 

Because the simulation provides the 8-Stage Engine with infinite "free energy" to run its unitary rotations and Lindblad projections, the Engine naturally maintains a lower internal entropy than the Rock. The Engine is effectively a free-rider, violating Bridge T.

## The Physical Solution
To model physical reality, Solvency MUST be a gradient function that penalizes active operations:
$\text{True Solvency} = \text{Terminal Negentropy} - \sum (\text{Landauer Work Costs})$

If we penalize the 8-Stage Engine by subtracting the true thermodynamic cost of its projections (calculated via `compute_landauer_cost`), its net solvency in a calm environment will drop below zero.

**Conclusion:** The Rock *must* win in calm environments because the structural utility generated by the 8 stages is outweighed by their thermodynamic maintenance cost. The simulation only failed to show this because it did not balance the thermodynamic checkbook.
