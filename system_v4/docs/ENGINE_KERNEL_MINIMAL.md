## 2. Engine-Family Grammar

- There are 2 engine families / 2 spinor-side engine types.
- Each engine has 2 loops.
- Each loop has 4 stages.
- Each engine therefore has 8 macro-stages.
- Each engine contains both a heating-role loop and a cooling-role loop.
- Left/right engine families invert where those loop roles sit.
- `engine_type` is not the same thing as:
  - flow direction
  - chirality/flux label
  - Axis 6 precedence
  - heating/cooling role
