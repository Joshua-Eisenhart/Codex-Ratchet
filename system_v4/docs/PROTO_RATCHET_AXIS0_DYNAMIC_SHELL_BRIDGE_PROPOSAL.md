## 4. Dynamic Shell Field

The shell layer is not a rigid nested-container model.

The proposal is that shells are:

- dynamic
- pulsing
- stretching
- compressing
- information-bearing
- cut-bearing

The shells are not static partitions. They are moving interfaces on the carrier geometry across which information and correlation are redistributed.

Minimal abstract notation:

\[
\Sigma_r
\]

for the shell at discrete shell index \(r\).

Each shell carries a shell state or shell data object:

\[
\mathcal{S}_r
\]

and the shell field evolves by discrete updates such as:

\[
\mathcal{S}_r \mapsto \mathcal{S}_{r+1}
\]

for one outward tick, or

\[
\mathcal{S}_r \mapsto \mathcal{S}_{r-1}
\]

for one inward tick.

The intended physical reading is:

- outward shells can carry retained information
- inward shells can represent converging possibility structure
- the present is not a static shell, but a convergence / compression interface

This is still a proposal, not a closed theorem.

## 5. Discrete Tick Rule

This proposal rejects infinite shell divisibility as primitive.

The shell ladder is:

- finite
- discrete
- tick-based

At some level, each outward or inward tick yields a distinct shell:

\[
r \in \{0,1,2,\dots,R\}
\]

for some finite horizon \(R\), or for a finite active window if the total horizon is model-dependent.

So the bridge should be formulated using finite differences and finite shell updates, not infinitesimal shell-radius derivatives:

\[
\Delta i_r = i_{r+1} - i_r
\]

or

\[
\Delta \Phi_{0,r} = \Phi_{0,r+1} - \Phi_{0,r}.
\]

This fits the repo-wide finitude discipline better than a continuum-first shell ontology.

- each shell corresponds to a discrete tensor-network layer or cut
- shell compression corresponds to coarse-graining / contraction / rank control
- shell expansion corresponds to refinement / branching / larger admissible completion space
- shell information is carried by entanglement structure across cuts

Minimal abstract objects:

\[
\mathcal{T}_r
\]

for a shell-layer tensor object, and

\[
\chi_r
\]

for shell-cut bond or entanglement structure at layer \(r\).

Candidate shell observables include:

\[
S_{\mathrm{cut}}(r),
\]

\[
I(A_r:B_r),

\[

or a path / branch entropy over admissible shell refinements.

This does not force tensor networks into canon. It only identifies them as a strong mathematical language for dynamic, compressive, discrete shell structure.
